# Contributing to Awesome AI Resources

Thanks for helping make this the best AI resources list on GitHub!

## Ground Rules

- **No AI-generated PRs.** We review every submission manually. PRs that appear to be mass-generated by AI will be rejected.
- **Be specific.** Don't add something just because it has stars. Explain why it belongs here.
- **Quality > Quantity.** We'd rather have 100 excellent entries than 1,000 mediocre ones.

## How to Contribute

### Option 1: Open an Issue (Recommended for Discussion)

Use the **Resource Suggestion** template to propose a new entry. This lets the community discuss it before you spend time writing a PR.

### Option 2: Direct Pull Request

1. **Fork** the repository
2. Create a branch: `git checkout -b add-awesome-thing`
3. Add your entry in the correct category
4. **Update the Table of Contents** if you added a new section
5. Commit with a clear message: `Add [project name] to [category]`
6. Open a PR against `main`

## Entry Format

```markdown
| 项目名 | 作者 | 描述 | 语言/平台 | Star | [GitHub](https://github.com/owner/repo) |
```

### Good Descriptions

- ✅ "Universal memory layer for AI agents — persistent, searchable, self-improving"
- ✅ "Multi-agent conversation framework with role-based collaboration"
- ✅ "Chinese curated list of production-ready MCP servers"

### Bad Descriptions

- ❌ "An awesome tool"
- ❌ "The best framework for AI agents"
- ❌ "Check it out!! 🚀🔥"

## Acceptance Criteria

| Criteria | Threshold |
|----------|-----------|
| ⭐ Stars | ≥ 100 (exceptions for niche, high-quality tools) |
| 🕐 Last commit | ≤ 6 months ago |
| 📝 Description | Included and useful |
| 🏷️ Category | Correctly placed |
| 🔗 Links | Working and direct to repo |

## What We Don't Accept

- 🚫 Closed-source / paid-only tools (open-source or freemium only)
- 🚫 Abandoned projects (no commits for > 1 year)
- 🚫 Duplicate entries
- 🚫 Self-promotion without disclosure
- 🚫 Spam or low-effort links

## Category Guidelines

If you're unsure where to place an entry:

- **MCP Servers** → Tools that implement the MCP server specification
- **MCP Clients** → Apps that connect to MCP servers
- **Agent Skills** → Reusable SKILL.md or plugin modules for AI agents
- **AI Agent Frameworks** → Libraries for building AI agents
- **Tools & Platforms** → AI-enhanced products and services
- **Tutorials & Courses** → Educational content
- **Best Practices** → Guides, checklists, and design patterns

## Review Process

1. Automated checks run on every PR (link validation, format checking)
2. At least one maintainer reviews each PR
3. We may request changes or ask questions
4. Once approved, the PR is merged

We aim to review PRs within **1 week**.

## Adding a New Category

If you want to add a new category:

1. Open an Issue first to discuss
2. Explain why existing categories don't fit
3. Provide at least 5 entries that would go in the new category
4. We'll consider it!

## Translations

We maintain both English (`README.md`) and Chinese (`README-zh.md`) versions. When adding entries, please update **both files** if you're bilingual. If not, a maintainer will handle the translation.
